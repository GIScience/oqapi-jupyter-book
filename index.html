Willkommen zum ohsome Dashboard Tutorial!

OpenStreetMap (OSM) ist eine umfangreiche Datenbank nutzergenerierter, frei verfügbarer Geodaten für viele Anwendungen. Allerdings können unterschiedliche Datenqualität und fehlende Standardisierung die Nutzung erschweren. Deshalb haben wir das ohsome Dashboard entwickelt, damit du die Qualität der OSM-Daten weltweit oder für bestimmte Gebiete bewerten kannst und feststellen kannst, ob sie den Anforderungen deines Projekts entsprechen.

Dashboard URL

https://next.ohsome-dashboard.heigit.org/de

Das ohsome Dashboard ermöglicht die Analyse von OSM-Daten mit vollständiger Versionshistorie - ganz ohne Programmierkenntnisse. Es erstellt Statistiken und visualisiert diese direkt, mit anpassbaren Filtern und Gruppierungen nach Tags (Kennzeichnungen), Typen, Regionen und Zeit. Die ohsome Quality API bietet Zugriff auf Informationen zur Qualität von OSM-Daten für spezifische Regionen und Anwendungsfälle.

Datenqualität analysieren im ohsome Dashboard

  1. Analyseregion wählen

  2. OSM Datenthema wählen

  3. Qualitätsindikator wählen

  4. Ergebnis auswerten

Dieses Tutorial wurde in Zusammenarbeit mit dem Bundesamt für Kartographie und Geodäsie (BKG) erstellt. Mehr Informationen zum Projekt „Qualitätsabschätzung von OpenStreetMap-Daten“ finden sich auf der Projektwebsite.
